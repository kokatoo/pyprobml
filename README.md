# pyprobml (R translation)

This repository contains R translations of figures and models from Kevin Murphy’s *Probabilistic Machine Learning: An Introduction* (“Book 1”) and *Advanced Topics* (“Book 2”), based on the original Python implementation.

The original codebase is written in Python using NumPy, SciPy, Matplotlib, scikit-learn, and more. This project reimplements selected notebooks in R, primarily using `tidyverse` and `ggplot2`.

The goal is to provide a faithful R-based counterpart to the original Python notebooks.
